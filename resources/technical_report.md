# SAR-EO Translation

## Abstract
This report addresses the problem of translating synthetic aperture radar (SAR) imagery into electro optical imagery using deep generative adverserial networks. The proposed approach uses a paired image-to-image translation pipeline with task-specific preprocessing, a hybrid L1-GAN loss, and controlled ablations to evaluate the contribution of each modeling decision. Across both evaluation splits, the method is trained and evaluated on Sentinel-1&2 Image Pairs (SAR & Optical) using perceptual metrics((LPIPS) and Frechet inception distance (FID)) and pixel-level metrics(PSNR and SSIM), along side qualitative triplets. The report also discusses limitations, error patterns, and concrete next-step directions for model, data, and training improvements.
